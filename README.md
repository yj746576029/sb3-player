# sb3-player
- 你只需把你的sb3文件命名为demo.sb3,替换根目录下的resources\app\demo.sb3即可
- 如果你下载的是发行版zip文件，首先解压zip文件，然后把你的sb3文件命名为demo.sb3,替换根目录下的resources\app\demo.sb3，然后双击运行sb3-player.exe即可

```bash
# 安装
npm install
# 运行web
npm run serve
# 运行electron
npm run electron:serve
# web打包
npm run build
# electron打包
npm run electron:build
```
